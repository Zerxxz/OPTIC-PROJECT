Deploy in three commands

From a fresh clone to a live testnet agent.

COMMAND 1

Run the tests

Verify the build is green before deploying.

sui move test                          # 38/38
npx tsx --test src/__tests__/*.test.ts # 12/12
COMMAND 2

Publish

Build and publish the Move package to Sui. The script captures the new PACKAGE_ID and writes it to sdk/src/constants.ts and orchestrator/.env.

./scripts/publish.sh --network testnet
COMMAND 3

Initialize the agent

One PTB: create treasury, create agent, mint 3 caps, transfer all to the active address. The script captures agent + treasury ids back to .env.

./scripts/init-agent.sh \
  --name agent-alpha \
  --strategy mean-reversion
COMMAND 4

Run the demo

Watch a 14-cycle orchestrator run with a moving market. Each line is a real Decision from the on-chain-style state machine.

npx tsx scripts/demo-cycle.mts --n 14
